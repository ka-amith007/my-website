HOW TO ADD PROJECT SCREENSHOTS
================================

Please add screenshots of your projects in this folder with these exact names:

1. disease-prediction.png    - Screenshot of Multiple Disease Prediction app
2. route-optimizer.png       - Screenshot of AI Route Optimization app  
3. crop-recommendation.png   - Screenshot of Crop Recommendation System
4. sign-language.png         - Screenshot of Sign Language Converter
5. weather-app.png           - Screenshot of Weather Website
6. task-manager.png          - Screenshot of Task Manager Application

HOW TO TAKE SCREENSHOTS:
-------------------------
1. Open each project's live demo link
2. Take a screenshot (Windows Key + Shift + S)
3. Save it in this folder with the correct name
4. Recommended size: 1200x600 pixels or similar aspect ratio

The website will automatically use these images once you add them!
If an image is not found, it will show a beautiful gradient background with an icon as fallback.
